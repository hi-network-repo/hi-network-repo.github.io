**自分で用意するの物**

- 熱中症対策用品（水分は多めに持参ください） 

- 日焼け対策になる服装（半袖半ズボン，サンダルは不可） 

- 日焼け止め(何でもOK)

- 帽子，サングラス必須 

- おやつ（塩味を含んだものがあると良い） 

- 酔い止め薬 

- 1dayレジャー保険（必ず加入して下さい） 

- クーラーボックス，氷 

**船長が用意する物**

- ライフジャケット（船内備え付け） 

- 釣竿，リール，仕掛け，エサ 

- 洗い用の水
